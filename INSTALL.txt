================================================================================
                            TROUBLESHOOTING
================================================================================

PROBLEM: "npm is not recognized"
SOLUTION: Install Node.js from https://nodejs.org/

PROBLEM: "Port 3000 is already in use"
SOLUTION: Use a different port:
npm run dev -- -p 3001
Then open: http://localhost:3001

PROBLEM: Installation fails
SOLUTION: Delete node_modules folder and try again:
rmdir /s node_modules (Windows)
rm -rf node_modules (Mac/Linux)
npm install

PROBLEM: Website doesn't load
SOLUTION: 
1. Check if server is running (you should see "Ready" message)
2. Try refreshing browser
3. Clear browser cache

OPTION 1: Vercel (Easiest - FREE)

1. Create account at vercel.com
2. Install Vercel CLI: npm install -g vercel

OPTION 2: Netlify (FREE)
-------------------------
1. Create account at netlify.com
2. Drag and drop .next folder
3. Your site is live!

OPTION 3: Manual
----------------
1. Run: npm run build
2. Upload .next folder to your hosting
